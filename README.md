# Natale 3D con Streamlit

App web di auguri di Natale costruita con Streamlit e Plotly. Mostra un albero 3D con livelli di verde, palline colorate, lucine, pacchi regalo e un messaggio romantico personalizzabile. La scena 3D è ottimizzata per essere esplorata su telefono (pinch-to-zoom, rotazione orbitale con il dito) e su desktop.

## Come eseguirla in locale
1. Clona la repository.
2. (Opzionale) Crea e attiva un ambiente virtuale Python.
3. Installa le dipendenze: `pip install -r requirements.txt`.
4. Avvia l'app: `streamlit run app.py`.

## Come pubblicarla su Streamlit Community Cloud
1. Carica questa repository su GitHub.
2. Vai su [Streamlit Community Cloud](https://streamlit.io/cloud) e crea una nuova app collegando il repository.
3. Seleziona `app.py` come file principale e distribuisci: l'app sarà online in pochi minuti.

## Come vederla subito dal telefono
1. Dopo la pubblicazione su Streamlit Community Cloud copia l'URL pubblico generato (è del tipo `https://tuo-progetto.streamlit.app`).
2. Apri quell'URL direttamente dal browser del telefono e usa le gesture: trascina per orbitare intorno all'albero e fai pinch-to-zoom per avvicinarti.
3. Se vuoi condividere l'esperienza, invia lo stesso link via chat: l'app gira nel cloud e non richiede PC né installazioni.
